# Grain (小麦) - Product Overview

Grain is a modern, cross-platform writing application designed for long-form writing, knowledge management, and creative expression.

## Core Features

- **Rich Text Editor**: Lexical-based editor with Markdown shortcuts
- **Workspace Organization**: Tree-based file management with workspaces, folders, and files
- **Search**: In-file search/replace, global full-text search with regex support
- **Icon Themes**: 6 preset themes (VSCode-style icon theming)
- **Outline & Charts**: Mermaid and PlantUML diagram support
- **Tag System**: Content organization with `#[tags]`
- **Wiki Links**: Quick references via `@` symbol with hover preview
- **Daily Journal**: Built-in diary system
- **Export**: JSON, ZIP, Markdown, DOCX formats
- **Offline Storage**: IndexedDB + Dexie for local persistence

## Target Platforms
