## 设计原则

1. **一致性**：所有组件使用统一的圆角、间距、配色
2. **层次感**：通过阴影、渐变、透明度营造深度
3. **柔和性**：避免生硬的直角和纯色
4. **可读性**：优化字体、配色、对比度
5. **响应性**：添加悬停、点击等交互反馈

## 技术细节

- 使用 Tailwind CSS 的现代化工具类
- 利用 CSS 变量实现主题适配
- 使用 `color-mix()` 实现半透明效果
- 添加 CSS 过渡动画提升体验
- 优化深色模式下的配色
