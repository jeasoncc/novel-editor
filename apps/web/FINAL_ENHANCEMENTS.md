## 🎯 使用指南

### 替换截图

1. 准备应用截图（建议尺寸：1920x1080 或类似比例）
2. 将截图放在 `public/screenshots/` 目录
3. 更新 `screenshots-section.tsx` 中的图片路径

### 更新统计数据

在 `stats-section.tsx` 中更新真实数据：
```typescript
{
  value: "10K+",  // 更新为真实数字
  label: "活跃用户",
  // ...
}
```

### 添加更多 FAQ

在 `faq-section.tsx` 的 `faqs` 数组中添加：
```typescript
{
  question: "新问题？",
  answer: "详细回答...",
}
```
