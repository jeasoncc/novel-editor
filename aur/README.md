## Installation

### From AUR
```bash
yay -S novel-editor
# or
paru -S novel-editor
```

### Manual Installation
```bash
git clone https://aur.archlinux.org/novel-editor.git
cd novel-editor
makepkg -si
```

## System Requirements

- **OS**: Arch Linux (x86_64)
- **Dependencies**: webkit2gtk, gtk3, libappindicator-gtk3
- **RAM**: 4GB recommended
- **Storage**: 500MB free space

## Usage

Launch from application menu or run:
```bash
novel-editor
```
