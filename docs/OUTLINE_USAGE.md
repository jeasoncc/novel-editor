# 大纲功能使用指南

## 🎯 快速开始

### 1. 访问大纲视图

有两种方式访问大纲功能：

**方式一：侧边栏按钮**
- 在左侧活动栏找到"大纲"图标（📋 ListTree）
- 点击进入大纲视图

**方式二：直接访问**
- 在浏览器地址栏输入：`/outline`

### 2. 视图切换

大纲提供两种主要视图模式：

#### 🌲 树形视图（推荐）
- **适用场景**：整体结构规划、快速导航
- **特点**：
  - 清晰的层级结构
  - 支持折叠/展开章节
  - 实时字数统计
  - 状态图标显示

#### 📇 卡片视图
- **适用场景**：内容概览、视觉化管理
- **特点**：
  - Scrivener 风格的卡片布局
  - 显示场景摘要（前100字）
  - 按章节分组展示
  - 响应式网格布局

### 3. 搜索功能

使用搜索框快速定位内容：
- 输入章节名称或场景名称
- 实时过滤显示结果
- 支持模糊匹配

## 🔧 功能详解

### 树形视图操作

#### 章节节点
- **展开/折叠**：点击章节前的箭头图标
- **查看信息**：
  - 章节标题
  - 包含的场景数量
  - 总字数统计
- **操作菜单**：鼠标悬停显示操作按钮
  - 添加场景
  - 重命名章节
  - 删除章节

#### 场景节点
- **点击跳转**：点击场景节点直接跳转到编辑器
- **状态图标**：
  - ⭕ 空圆圈 = 草稿（0字）
  - ✅ 绿色勾 = 已完成（有内容）
- **字数显示**：实时显示场景字数
- **操作菜单**：
  - 重命名场景
  - 复制场景
  - 移动到其他章节
  - 删除场景

### 卡片视图操作

#### 章节分组
- 每个章节显示为一个独立的卡片组
- 显示章节标题和场景数量徽章

#### 场景卡片
- **卡片内容**：
  - 场景标题
  - 字数统计
  - 内容摘要（前100字）
- **卡片样式**：
  - 普通卡片：有内容的场景
  - 虚线边框：空白场景（草稿状态）
- **交互**：
  - 点击卡片跳转到编辑器
  - 悬停显示操作菜单

## 📊 统计信息

底部状态栏显示项目统计：
- 总章节数
- 总场景数
- 总字数

## 💡 使用技巧

### 1. 规划创作流程
```
树形视图 → 搭建大纲结构
   ↓
卡片视图 → 查看整体内容概览
   ↓
点击场景 → 开始具体写作
```

### 2. 快速导航
- 使用搜索功能快速定位章节/场景
- 在树形视图中折叠无关章节，专注当前部分
- 点击场景直接跳转编辑，无需多级导航

### 3. 进度追踪
- 通过状态图标快速识别完成情况
- 字数统计帮助了解各章节/场景的篇幅
- 底部总字数跟踪项目整体进度

### 4. 结构调整（即将上线）
- 拖拽章节调整顺序
- 拖拽场景在章节间移动
- 批量操作多个节点

## ❓ 常见问题

### Q: 如何添加新章节？
A: 目前需要在主编辑页面添加章节，大纲视图会自动同步。未来版本将支持在大纲中直接添加。

### Q: 可以在大纲中编辑场景内容吗？
A: 大纲视图主要用于结构化管理。点击场景节点/卡片会跳转到编辑器进行内容编辑。

### Q: 字数统计实时更新吗？
A: 是的。当你在编辑器中修改内容并保存后，返回大纲视图会看到最新的字数统计。

### Q: 支持多项目切换吗？
A: 当前大纲视图默认显示第一个项目。未来版本将支持项目选择器。

### Q: 拖拽功能什么时候上线？
A: 拖拽重排功能已在开发路线图中，预计下个版本推出。
