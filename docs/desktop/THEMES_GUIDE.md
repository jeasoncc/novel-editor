## 🔧 如何切换主题

### 方法 1: 设置页面
1. 打开设置（点击 ActivityBar 底部的齿轮图标）
2. 进入"外观"或"主题"设置
3. 从下拉列表中选择主题
4. 主题立即应用

### 方法 2: 命令面板
1. 按 `Ctrl+K` (Mac: `Cmd+K`) 打开命令面板
2. 输入 "theme" 或"主题"
3. 选择"切换主题"命令
4. 从列表中选择主题

### 方法 3: 快捷键（如果配置）
- 可以在设置中配置快捷键快速切换主题

## 📚 技术细节

### 主题结构
```typescript
interface Theme {
  key: string;           // 唯一标识
  name: string;          // 显示名称
  description: string;   // 描述
  type: "light" | "dark"; // 类型
  colors: ThemeColors;   // 颜色配置
}
```

### 颜色配置
```typescript
interface ThemeColors {
  // 基础颜色
  background: string;
  foreground: string;
  
  // 主色调
  primary: string;
  primaryForeground: string;
  
  // 侧边栏
  sidebar: string;
  sidebarForeground: string;
  
  // ... 更多颜色
}
```

### 应用主题
```typescript
import { applyTheme, getThemeByKey } from "@/lib/themes";

// 获取主题
const theme = getThemeByKey("tokyo-night");

// 应用主题
if (theme) {
  applyTheme(theme);
}
```
