# 图标设置页面 - 预览窗口更新

## ✅ 更新完成

已将图标设置页面重新设计为与 Appearance 页面相同的展示方式，左右布局，右侧有虚拟窗口实时预览图标效果。

## 🎨 新的页面布局

### 左侧（5/12 列）
- **图标主题选择**
  - 6 个主题卡片（2 列网格）
  - 每个卡片显示 3 个代表性图标
  - 简洁的主题信息
  - 选中状态高亮

### 右侧（7/12 列）
- **虚拟预览窗口**
  - 模拟应用窗口（带红黄绿按钮）
  - 左侧：文件树预览（显示文件图标）
  - 右侧：ActivityBar 图标预览
  - 实时响应主题切换

## 📱 预览窗口内容

### 左侧文件树（宽度 224px）
显示完整的项目结构，包含：
- 📖 项目图标
- 📁 章节图标（支持展开/收起）
- 📄 场景图标
- 📂 文件夹图标
- 👤 角色图标
- 🌍 世界观图标

**特点**：
- 层级缩进显示
- 当前选中项高亮
- 使用当前颜色主题的侧边栏颜色

### 右侧 ActivityBar 预览
显示 9 个主要的 ActivityBar 图标：
- 书库、搜索、大纲
- 角色、世界观、绘图
- 统计、设置、新建

**特点**：
- 3 列网格布局
- 每个图标带标签
- 悬停效果
- 使用当前颜色主题的背景色

## 💡 使用提示

1. **选择主题**：点击左侧任意主题卡片
2. **查看效果**：右侧预览窗口立即更新
3. **文件图标**：在左侧文件树中查看
4. **ActivityBar 图标**：在右侧网格中查看

## 🚀 技术实现

### 响应式设计
- 大屏幕：左右布局（5:7 比例）
- 小屏幕：上下堆叠

### 主题集成
- 使用 `useTheme()` 获取当前颜色主题
- 预览窗口使用主题颜色
- 实时响应主题变化

### 状态管理
- 使用 `useState` 管理当前图标主题
- 使用 `useEffect` 监听主题变化事件
- 自动更新预览内容
